# 公司内部包使用指南

本目录（41~60）预留用于收录公司内部自研包/框架的使用规范。

---

# 一、收录规则

每个内部包单独一个文档，编号从 41 开始递增：

```
41-xxx-sdk.md          — 示例：内部 SDK 使用规范
42-xxx-framework.md    — 示例：内部框架使用规范
43-xxx-common.md       — 示例：内部公共组件使用规范
...
60-xxx.md              — 最多 20 个文档
```

## 文档命名规范

```
{编号}-{包简称}.md
```

- 包简称使用小写英文，多个单词用连字符分隔
- 示例：`41-kunlun.md`、`42-taihong-framework.md`

---
