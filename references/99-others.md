# 一、适用场景

以下内容适合收录到本文档：

- 特定框架或中间件的使用规范（如 MyBatis、Redis、MQ 等）
- 特定业务场景的编码模式（如审批流、数据导入导出等）
- 特定协议或格式的处理（如 WebSocket、gRPC、Protocol Buffers 等）
- 团队特有的约定或惯例
- 其他不便归类的实用技巧

---

# 二、收录规则

1. 每条规则以 `#` 一级标题或 `##` 二级标题分组
2. 同类规则归入同一章节
3. 格式参照 `02-code-examples-template.md` 中的写入模板
4. 随着内容增多，可考虑将成熟章节独立为新文档
